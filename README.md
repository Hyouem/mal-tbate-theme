# TBATE MyAnimeList Theme — Final

Este repositório contém somente os arquivos necessários para a versão final.

## Estrutura

- `assets/arthur-background-widescreen.jpg`
- `TBATE_MAL_FINAL.css`
- `TBATE_MAL_FINAL.txt`
- `README.md`

## Publicação

Crie ou recrie o repositório público:

`Hyouem/mal-tbate-theme`

Envie estes arquivos mantendo a pasta `assets` e a branch `main`.

O CSS usa esta imagem:

`https://raw.githubusercontent.com/Hyouem/mal-tbate-theme/main/assets/arthur-background-widescreen.jpg`

## MyAnimeList

Copie todo o conteúdo de `TBATE_MAL_FINAL.txt` para o editor Advanced CSS e clique em `Update CSS`.

Depois recarregue a lista com `Ctrl + F5`.
